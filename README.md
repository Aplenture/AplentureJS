# AplentureJS
## Installation
### Latest version
```
npm install https://github.com/Aplenture/AplentureJS.git
```
### Specific version
```
npm install https://github.com/Aplenture/AplentureJS.git#v1.0.0
```
## Usage
### Server
1. Create a database update to create a new table.
1. Create a repository to implement database queries.
1. Create a command to implement a REST API.
1. Define a new repository or command in the config template.
### Client
1. Create a view controller to implement a new page.
1. Use request to implement a REST API.
1. Define the url of REST API in the config template.
